Basic HTML structure

(Voltar)

The HTML document always starts with what we call the basic structure. It is basically immutable, you will always repeat that it in your code:

"< head >": Contains information that is not visibly transposed to the user/reader of the document. It is implicit data, of use and control of the document. In practice, all header content is delimited between the opening and closing tag head.

"< body >":This is the document itself, that is, the information readable to the user/reader of the document. It is any text you want to present, as well as any form of output media (images, sounds, embedded applets, multimedia content, etc.). In addition, the entire data entry presentation (forms) also applies in this section of the document. In practice, the body of the document is delimited by the tag pair < body > and < /body >.

Tags

(Voltar)

Tags are used to inform the browser of the structure of the website. That is, when writing HTML code, the tags will be interpreted by the browser, thus producing the structure and visual content of the page.

Texts

(Voltar)

One of the main goals of HTML is to give text structure and meaning, also known as semantics), so that a browser can display it correctly. This article explains how HTML can be used to structure a text page, add headings and paragraphs, emphasize words, create lists, and more.
